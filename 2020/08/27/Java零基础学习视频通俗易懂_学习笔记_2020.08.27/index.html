【狂神说Java】ava零基础学习视频

P1 前言01：解决大家的疑问

聊聊 Java 这条路 - “即使再小的帆也能远航” 不管是什么情况只要足够的坚持

疑问1：零基础可能学习java吗？必须的。建议：学习是为了什么？要搞清楚，有明确的目标。是找工作、还是建一个网站、写一个游戏。

疑问2：英语不好能学吗？程序真的没有大家想象的那个多的英语，而且天天都在用，慢慢就掌握了！

疑问3：理解慢能学好吗？ Of course，编程需要一定的逻辑思维能力，但更多的是持续性的学习！

疑问4：现在学Java晚吗？有用吗？ 最好的时候就是现在，Java Python 其他什么技术都要会，Java未来的开发之路会非常宽！要看语言的生态！

P2 前言02：Java和python的抉择

学好了Java，其他语言可以在短时间学好，如果想做专业的程序员，从Java开始比较好！都要会！把编程当成一个爱好！

P5 预科01：博客的重要性

学习准备：博客

博客，英文名Blog，它的正式名称为网络日记

为什么要学博客？

  • 需要总结和思考。有时候我们一直在赶路，却忘记方面脚步 （是重要的一点，总结和思考）
  • 提升文笔组织能力
  • 提升学习总结能力
  • 提升逻辑思维能力
  • 帮助他人，结交朋友

冰冻三尺非一日之寒，写博客也是，短期内可能看不到效果，但是长期坚持，对自己的提升很有帮助！

P6 预科02：MarkDown语法详解

MarkDown语法

  • 推荐文本编辑器：Typora
  • 文件后缀 xxx.md
  • 样式：
    • 标题
      • #空格 一级标题
      • ##空格 二级标题
      • …. 最多到6级标题
    • 字体
      • 粗体：**内容**
      • 斜体：*内容*
      • 斜体加粗：***内容***
      • 删除线：~内容~
    • 引用 >空格+内容
    • 分割线 --- 或 ***
    • 图片 ![名字]() 括号中存放图片的路径，可以本地、网络两种
    • 超链接 [名字]() 括号中存放链接地址
    • 列表
      • 有序 1.空格
      • 无序 -空格 内容
    • 代码 ```
    • 表格
      • 可以直接插入（软件提供的）
      • 语法 名字｜性别｜写两行
name price count
